Frequently asked questions
Everything you need to know about agentic RAG.

What makes it "agentic"?

Unlike traditional RAG, our agents can plan multi-step workflows, use tools autonomously, and self-correct based on results.

Can I use my own vector database?

Yes! BYODB support for Pinecone, Weaviate, Qdrant, and more. We also offer managed vector storage.

How does pricing work?

Based on token usage and number of agent executions. Generous free tier included.

Is my data used for training?

Never. All data stays within your tenant. Enterprise plans offer complete data isolation.

What's the latency like?

Average response time is 1.8 seconds for complex queries. Streaming responses available for real-time feedback.

Can agents call external APIs?

Yes! Agents can be configured with tool access to REST APIs, GraphQL endpoints, and custom functions.
